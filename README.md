# karasubonk bilibili
一款支持B站直播的砸头插件。

![image](https://github.com/LuiScreaMed/karasubonk/assets/73584430/98a95f5e-7dce-468a-8ac4-0e8073ecb5b9)

修改自原作者 [Typeou](https://github.com/typeou/karasubonk)，使之支持B站直播。

支持B站直播中的：
- 礼物
- 上舰
- 关注
- 点赞
- 弹幕关键字
